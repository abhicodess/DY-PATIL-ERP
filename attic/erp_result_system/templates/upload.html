Import Guidelines
Required File Format:

Ensure your Excel sheet contains the following exact headers for successful auto-mapping:

  • PRN Unique ID
  • Name Student Name
  • Division A, B, C, or D (Optional)
Subject Columns Layout (Wide Format):

Marks must be mapped dynamically grouped by Subject Code postfixes.

SUB1_Assignment, SUB1_Attendance, SUB1_UT, SUB1_MSE,
SUB2_Assignment, SUB2_Attendance, SUB2_UT, SUB2_MSE...
Note: Out-of-bounds components handles missing datasets as 0 marks gracefully. Marks should be relative to Assignment (5), Attendance (5), UT (20) and MSE (20) making each subject precisely aggregate to 60.
