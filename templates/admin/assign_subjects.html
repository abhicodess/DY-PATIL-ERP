How Faculty Attendance Works — 3 Steps
Step 1 — Assign Subject

Set the subject's Teacher field to the faculty's exact name (e.g. Prof.Shakil Tamboli). This page does that automatically.

Step 2 — Send Student List

Send the class student list to the faculty's Messages inbox so they know which students to mark.

Step 3 — Faculty Marks

Faculty logs in → Mark Attendance → selects their subject from dropdown → marks P/A/L/M for each student.
